## Installation

To get started with ReSense AI, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/Bahaa202/resense-ai.git
    ```
2. Navigate to the project directory:
    ```bash
    cd resense-ai
    ```
3. Install the required dependencies:
    ```bash
    npm install
    ```
